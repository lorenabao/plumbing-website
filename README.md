# plumbing_website

Workspace for plumber website projects.

## Projects

- `arturo-morgadanes/` — Next.js (App Router) website + docs in `arturo-morgadanes/docs/`
- `assets/` — supporting assets (e.g. prompt PDF)

## Quick start (Arturo Morgadanes)

```bash
cd arturo-morgadanes
npm install
cp .env.local.example .env.local
npm run dev
```
